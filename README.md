# ELEV8 Media — Operations App

Internal daily ops tracker for the Elev8 Media team.

## Setup

### 1. Environment variables
Create a `.env.local` file in the root with:

```
VITE_SUPABASE_URL=https://jnqisiwhqzjhqrjexkvu.supabase.co
VITE_SUPABASE_ANON_KEY=your_anon_key_here
VITE_MANAGER_PASSWORD=Elev8Admin
```

### 2. Install and run locally
```
npm install
npm run dev
```

### 3. Deploy to Vercel
- Connect this GitHub repo to Vercel
- Add the environment variables in Vercel dashboard
- Deploy
